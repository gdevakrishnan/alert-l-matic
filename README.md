# Alert-L-Matic
Alert-L-Matic is an application for landslide disaster management system. 
It was working with the both Hardware and Software.

## Technology
### Hardware
* Microcontroller
* LoRa

### Software
* React Native
* Nodejs
* Python
* Hardhat
* Solidity
